# 简约音乐播放器

这是一个 JavaFX 写的本地音乐播放器，支持从文件夹导入歌曲、播放控制、播放顺序切换，以及本地/联网歌词显示。

## 功能

- 从文件夹递归导入歌曲
- 支持常见音频格式：`mp3`、`m4a`、`aac`、`wav`、`aif`、`aiff`
- 显示歌曲名、歌手名
- 播放模式：顺序播放、随机播放、单曲循环
- 在播放列表中搜索歌曲、歌手或文件名
- 应用界面获得焦点时，按空格键播放/暂停
- 优先读取歌曲同目录同名 `.lrc` 歌词
- 没有本地歌词时，可通过 LRCLIB 联网搜索歌词
- 歌词未找到时会在后台持续重试，直到找到或切换歌曲
- 使用本地 SQLite 数据库缓存已导入歌曲和已找到歌词
- 简约深色界面

## 运行

在 PowerShell 中进入项目目录后运行：

```powershell
.\run.ps1
```

第一次运行会自动下载 Gradle 到项目内的 `.tools` 文件夹，然后下载 JavaFX 依赖并启动程序。

## 歌手名识别

导入时会优先按文件名 `歌手 - 歌名.mp3` 识别歌手和歌名。播放后如果音频文件本身带有元数据，会自动用文件内的标题和歌手更新界面。

## 本地数据

程序会在项目目录下创建 `music-player.db`，用于保存导入过的歌曲和歌词缓存。下次启动时会自动恢复仍然存在的歌曲文件。
